# 🎓 UE Academic Course Registration System

A modern Flask-based Course Registration System developed as part of the Cloud Computing course at the University of Europe for Applied Sciences.

The application is deployed on an Amazon EC2 instance and allows users to manage students, professors, courses, and course registrations through a clean web interface.

---

## 🚀 Features

- Student Management
  - View all students
  - Add new students
  - Unique Student IDs (ST001, ST002, ...)

- Professor Management
  - View professor information

- Course Management
  - Browse available courses
  - Add new courses

- Course Registration
  - Register students for courses
  - View all registrations

- Modern responsive user interface

- AWS EC2 deployment
